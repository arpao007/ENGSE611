ตัวอย่างcss selectors

ข้้อความนี้ถูกจัดสีด้วย Element Selector (p { color: green; })

ข้อความนี้ถูกจัดสีด้วย Class Selector (.highlight { background-color: yellow; })

ข้อความนี้ถูกจัดสีด้วย ID Selector (#header { background: blue; color: white; })

Pseudo-class :hover

นำเมาส์ชี้ที่ข้อความนี้เพื่อดูการเปลี่ยนแปลงสี (p:hover { color: red; background-color: #ffe5e5; })
